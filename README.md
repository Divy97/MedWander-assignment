# MedWander-assignment

**Clone the repository:**

   ```bash
   git clone https://github.com/Divy97/MedWander-assignment.git
   ```

## Frontend
  ```bash
  cd client
  npm install
  npm run dev
  ```
## backend 
  ```
  cd server
  npm install 
  ```

  Change host, user and password in **server.js**

  ```
  host: "localhost",
  user: "your_username",
  password: "your__password",
  ```
  After changing start server 

  ```
  node server.js
  ```
